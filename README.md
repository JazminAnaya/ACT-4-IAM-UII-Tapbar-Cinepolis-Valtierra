## Promt
Lenguaje Flutter Dart Nivel princi´piante. Ayudame a crear un tapbar inferior todo referido al negocio "Cinepolis Valtierra" utiliza colores azules rey, blancios y amarillos.
-En cada opción (tab) un icono acorde a la pantalla (Peliculas, Sucursales, Alimentos, Empleados y Clientes). Son cinco pantallas, en cada pantalla una imagen desde la red de github (Pantalla peliculas: https://raw.githubusercontent.com/JazminAnaya/ACT-4-IAM-UII-Tapbar-Cinepolis-Valtierra/refs/heads/main/peliculas.jfif, Pantalla Sucursales: https://raw.githubusercontent.com/JazminAnaya/ACT-4-IAM-UII-Tapbar-Cinepolis-Valtierra/refs/heads/main/sucursales.jfif, Pantalla Alimentos: https://raw.githubusercontent.com/JazminAnaya/ACT-4-IAM-UII-Tapbar-Cinepolis-Valtierra/refs/heads/main/alimentos.jfif, Pantalla Empleados: https://raw.githubusercontent.com/JazminAnaya/ACT-4-IAM-UII-Tapbar-Cinepolis-Valtierra/refs/heads/main/empleados.jfif, Pantalla Clientes: https://raw.githubusercontent.com/JazminAnaya/ACT-4-IAM-UII-Tapbar-Cinepolis-Valtierra/refs/heads/main/cliente.jpg)
